# Revenant-Spear
A more visible UI element for Revenant's Spear Abyssal Raze Mechanic

Tracks the stack count of Abyssal Raze and, if the duration is under 5 seconds, will also display it's timer.

# How to use
First download the zip file to your reffect root folder like so:

<img width="499" height="209" alt="image" src="https://github.com/user-attachments/assets/0c3853c6-ae9f-4739-9f20-5b3b8ac90b1b" />

Then right click > Extract here (the necessary files will be added to the packs and icons subfolders).

Then, return to reffect ingame and refresh packs.
